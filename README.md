# Yuan Tools

`yuan-tools` 是 Yuan 的小工具总索引仓库。

这里不直接存放每一个小工具的完整源码，而是作为所有已公开小工具的入口目录。每个工具都有自己的独立 GitHub 仓库、版本 tag 和 GitHub Release。

## 发布结构

每个小工具通常会提供两个交付版本：

- Web 静态版：`release/html`
- Windows 桌面版：`release/win`
